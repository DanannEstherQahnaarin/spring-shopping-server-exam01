## 2. 백업 및 복구

### 데이터베이스 백업

```bash
# MySQL 덤프
mysqldump -u user -p shopping > backup_$(date +%Y%m%d).sql

# 자동 백업 스크립트 (cron)
0 2 * * * /path/to/backup.sh
```

### 복구 절차

1. 최신 백업 확인
2. 데이터베이스 복구
3. 데이터 무결성 검증
4. 애플리케이션 재시작
