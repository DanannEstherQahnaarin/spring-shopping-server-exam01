# 아키텍처 결정 기록 (ADR)

## ADR-001: JWT 토큰 기반 인증 선택

### 상태

승인됨

### 배경

세션 기반 인증 vs 토큰 기반 인증

### 결정

JWT 토큰 기반 인증 채택

### 이유

- 서버 확장성 (Stateless)
- 마이크로서비스 아키텍처와의 호환성
- 모바일 앱 지원 용이

### 대안 고려

- OAuth 2.0: 복잡도 증가로 제외
- 세션 기반: 확장성 문제로 제외
