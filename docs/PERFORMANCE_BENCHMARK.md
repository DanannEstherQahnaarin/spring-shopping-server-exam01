# 성능 벤치마크

## 1. 부하 테스트

### JMeter 스크립트 예시

- 동시 사용자: 100명
- Ramp-up 시간: 10초
- 테스트 지속 시간: 5분

### 목표 지표

- 응답 시간: P95 < 500ms
- 에러율: < 0.1%
- TPS: > 1000

## 2. 성능 프로파일링

### JProfiler 활용

- CPU 사용률 분석
- 메모리 할당 패턴 분석
- 메서드 실행 시간 분석
